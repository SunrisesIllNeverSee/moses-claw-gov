## VI. THE VERIFICATION PROTOCOL

### Step 1: Establish the Series Baseline
Measure individual module viability under isolation.  
Calculate P(series) = ∏ P(mᵢ).  
This is your control value.

### Step 2: Apply Coupling Pressure
Engage all modules simultaneously under escalating task complexity.  
Do not use fixed-depth or fixed-load tests — constitutional systems respond dynamically.  
Load must stress the coupling architecture, not individual modules in isolation.

### Step 3: Measure Observed System Viability
Record P(observed) across the full coupled system under load.  
Use task performance, coherence retention, and recursion stability as viability proxies.

### Step 4: Calculate the Inversion Coefficient
```
IC = P(observed) / P(series)
```

### Step 5: Apply Detection Threshold
```
IC > 1.0  →  Constitutional governance confirmed
IC ≤ 1.0  →  Mimic detected
```
